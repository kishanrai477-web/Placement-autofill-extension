# 🚀 Placement Autofill Chrome Extension

## 📌 Overview

The **Placement Autofill Chrome Extension** is a productivity-focused browser extension designed to automate repetitive form-filling during job and internship applications.

It enables users to input their details once and automatically populate fields across job portals and Google Forms, significantly reducing manual effort and time.

---

## ✨ Key Features

* ⚡ **Automated Form Filling** – Detects and fills input fields intelligently
* 🧾 **User Data Management** – Input and manage details via popup UI
* 🌐 **Cross-Platform Support** – Works on Google Forms and job portals
* 🎯 **Time Optimization** – Speeds up bulk job applications
* 🧩 **Lightweight & Fast** – Minimal performance overhead

---

## 🛠️ Tech Stack

* **JavaScript (Vanilla JS)** – Core logic and DOM manipulation
* **HTML5** – Popup interface
* **CSS3** – UI styling
* **Chrome Extension APIs** – Browser integration

---

## 🏗️ Architecture

The extension follows a modular structure:

* `manifest.json` → Defines extension configuration and permissions
* `content.js` → Injects scripts and handles autofill logic
* `popup.html` → User interface for input
* `popup.js` → Manages user data and triggers autofill
* `style.css` → UI styling
* `icon.png` → Extension icon

---

## ⚙️ Installation & Setup

### 1️⃣ Load Extension in Chrome

1. Open Chrome
2. Navigate to: `chrome://extensions/`
3. Enable **Developer Mode** (top right)
4. Click **Load Unpacked**
5. Select the project folder

---

### 2️⃣ Usage

1. Open a job application form / Google Form
2. Click on the extension icon
3. Enter your details
4. Click **Autofill**
5. Form fields will be filled automatically
